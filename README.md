# Brick Blast
2D Game Using C++ Library SFML
## Prerequisites

Install SFML in Linux.

```bash
sudo apt-get install libsfml-dev

```

## Usage

### Linux
#### In Terminal Navigate To Game Folder
```bash
./build.sh
```
